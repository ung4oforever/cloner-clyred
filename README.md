## about

clyred server cloner is a fast discord server cloner written in python using **aiohttp** and **asyncio**. the project focuses on speed while preserving the original server structure as accurately as possible.

made by **zay (6cbj)**.

---

## features

```csharp
- fully asynchronous

- clones roles
  • name
  • color
  • permissions
  • hoist
  • mentionable
  • emoji
  • exact hierarchy order

- clones channels
  • text
  • voice
  • announcements
  • forums
  • stages
  • categories

- syncs permission overwrites

- remaps role ids automatically

- clones server settings
  • name
  • icon
  • description
  • locale
  • verification level
  • afk channel
  • system channel

- clones emojis

- automatic rate limit handling

- optimized for maximum speed
```
