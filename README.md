## 📌 About

**Clyred Server Cloner** is a high-speed Discord server cloner built with **Python** and **aiohttp**.

Designed to clone Discord servers asynchronously while respecting Discord rate limits for maximum performance.

Developed by **zay (6cbj)**.

---

## ✨ Features

```csharp
✔ Fully asynchronous (aiohttp)

✔ Fast Discord server cloning

✔ Clones exact role hierarchy

✔ Clones:
    • Name
    • Color
    • Permissions
    • Hoist
    • Mentionable
    • Role Emoji

✔ Clones channel types:
    • Categories
    • Text
    • Voice
    • Announcement
    • Stage
    • Forum

✔ Synchronizes permission overwrites

✔ Maps new role IDs automatically

✔ Clones server:
    • Name
    • Icon
    • Description
    • Locale
    • Verification Level
    • AFK Channel
    • System Channel

✔ Clones all emojis

✔ Automatic Discord 429 retry handling

✔ Lightweight & optimized
```
